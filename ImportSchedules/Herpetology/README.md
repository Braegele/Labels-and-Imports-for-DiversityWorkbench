# Section Herpetology

Data sheet for import

- New entries
- Features:
  - Format of taxon name: Genus  + Species epithet
  - Type status information
  - Separation of collectors
  - Revision data
  - "Not in collection" as transaction (transaction needs to be specified before import!)
